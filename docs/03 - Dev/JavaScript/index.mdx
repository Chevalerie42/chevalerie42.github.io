## Introduction 

JavaScript est un langage de programmation principalement utilisé pour les pages web. 
Il est souvent utilisé pour animer des pages Web, créer des effets, valider des formulaires, etc ...
JavaScript est interprété par le navigateur internet et peut être utilisé avec HTML et CSS pour créer des pages Web interactives. 
C'est un langage de programmation populaire et largement utilisé pour développer des applications web. Github constate même qu'en 2022, JavaScript soit en tête des langages les plus utilisés[^1]
