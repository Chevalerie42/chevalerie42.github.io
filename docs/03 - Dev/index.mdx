## Qu'est-ce que le developpement ?

Il s'agit de l'action de créer des programmes, des applications, des sites web ou des automatisations en utilisant différents langages de programmation. Pour se faire, le développeur écrit des lignes de code dans un fichier texte, et ainsi crée une application fonctionnelle ... ou presque :p 

## Que verrons-nous dans cette rubrique ?

Comment créer un script (ou un logiciel peut être ...) de A à Z. Nous apprendrons à coder avec différents langages tels que PowerShell, Bash, Python, et bien d'autres !
