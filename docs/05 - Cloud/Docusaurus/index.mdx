## Introduction 
<Term popup="CMS open source et gratuit permettant la génération de site statique spécialisé en documentation" reference="/glossaire/docusaurus">Docusaurus</Term> est un <Term popup="Content Management System (En français = Système de gestion de contenu), c'est un logiciel simple pour gérer un site web de A à Z avec peu ou pas du tout, de connaissances en développement" reference="/glossaire/CMS">CMS</Term> open source , gratuit, développé par la société Facebook.  
Il est conçu afin de permettre la création rapide et facile de sites web de documentation semblable à GitBook.  
La structure principale de <Term popup="CMS open source et gratuit permettant la génération de site statique spécialisé en documentation" reference="/glossaire/docusaurus">Docusaurus</Term> est codée en <Term popup="Il s'agit d'un langage de programmation principalement utilisé pour les pages web." reference="/glossaire/javascript">JavaScript</Term> et <Term popup="Une bibliothèque JavaScript permettant le développement des interfaces utilisateur interactives dans des applications web." reference="/glossaire/react">React</Term>, tandis que le contenu est rédigé en <Term popup="Le Markdown est un format de texte simplifié permettant de créer facilement du contenu structuré (en-têtes, listes, liens, etc ...)" reference="/glossaire/markdown">Markdown</Term>.  
En plus de cela, il offre plusieurs autres fonctionnalités telles que la gestion d'un blog et la mise en place d'un flux <Term popup="Really Simple Syndication (Français = Système de syndication simple), est un fichier qui permet de suivre les nouvelles publications d'un site. Il est très utilisé dans la veille technologique." reference="/glossaire/RSS">RSS</Term>.  
Contrairement à WordPress, <Term popup="CMS open source et gratuit permettant la génération de site statique spécialisé en documentation" reference="/glossaire/docusaurus">Docusaurus</Term> génère un site statique ce qui en assure une bonne vitesse de chargement.
