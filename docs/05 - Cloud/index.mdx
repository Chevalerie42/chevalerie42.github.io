## Qu'est-ce que le Cloud ?
Toutes les ressources informatiques (comme ce site web par exemple) qui sont disponibles via internet. Cela permet notamment de stocker, gérer ou, de manière plus générale, traiter des données sur des serveurs distants plutôt que sur son propre ordinateur.

## Que verrons-nous dans cette rubrique ?
Nous nous interesserons à tous les aspects liés au cloud. Cela inclura des sujets tels que la création d'un site WEB, l'utilisation de GitHub, ou encore des services de cloud tels qu'Azure ou AWS, entre autres.
